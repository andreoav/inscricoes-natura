## Sistema de Inscrições

O Sistema de inscrições é um projeto desenvolvido pelo [Natura Clube de Orientação](http://www.naturaco.org) com o objetivo de simplificar o gerenciamento de inscrições, tanto para os atletas quanto para os administradores do sistema.

## Desenvolvimento
Este sistema utiliza o framework [FuelPHP](http://www.fuelphp.com) como sua principal plataforma, o que possibilita um desenvolvimento ágil e organizado, seguindo os padrões de desenvolvimento mais aceitos no momento.

## Recursos
O Sistema visa simplificar as inscrições, abaixo estão os recursos que proporcionam esta simplificação:
* Perfil com todos os dados do Atleta sem a necessidade de inserir os dados a cada inscrição.
* Histórico completo de todas as inscrições realizadas pelo atleta.
* Guia de uso para novos cadastros com o objetivo de ensinar como usar o sistema.
* Lista de Etapas disponíveis para inscrição junto com o mapa e boletins informativos.
* Sistema de notícias integrado.
* Criação de relatórios de acordo com o modelo da CBO ou FGO (admin)
* Cadastro de campeonatos (admin)
* Cadastro de etapas (admin)
* Envio de email para todos os atletas cadastrado de maneira automática (admin)
* Entre outros...
